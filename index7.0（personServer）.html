Q：你们的退货承诺是什么？
A：尊敬的顾客，对于任何的退货我们承诺：
● 凡在本官方网站购买的商品未经穿着，请在24小时内与客服人员联系，提出需求，确认退货具体流程，并在15日内将商品寄回。
● 凡商品存在质量问题、发送（货）错误问题，支持无条件退换货，退换前请与客服人员联系，确认具体情况及流程，由此发生的邮寄费用由我公司承担。
● 在官方网站购买的商品暂不支持在实体店进行退货。同时，官方网站暂时也不受理实体店售出商品的退货申请。
● 如全部退货请将赠品以及发票一同寄回。
Q：我该怎么退货，具体需要注意什么？
A：尊敬的顾客，我司退货规则如下：
1. 退货申请必须在客户收到商品之日起（以签收日期为准）15日内，超过时间不予退货。
2. 商品存在质量问题，如破损、污渍、版型问题、商品缺失等，不包含色差、客户选购尺码不合适等问题。
3. 商品的外包装完好无损（包括包裹填充物及外包装箱或外包装袋），商品附件、吊牌、标签等齐全，商品保持官方网站出售时的原质原样不影响产品二次销售。
4. 发票（未开发票除外）、发货清单齐全。
5. 由于客户主观原因对商品不喜欢或感觉不符合心意的情况，符合退货条件可以办理，但需买家承担相应运费。

6. 暂不支持退货到付
以下情况不提供退货服务：
1. 超过退货规定时限的商品 ;
2. 已经穿着过或洗涤过的商品 ;
3. 已经修改或加工过的商品 ;
4. 直接接触皮肤的内衣类商品 ;
5. 商品吊牌、商标、票据等有缺失的商品 ;
6. 少量色差问题商品（发错颜色商品除外）

A：尊敬的顾客，关于积分商城的礼品，我们分为实物礼品以及虚拟礼品，具体政策如下：
适用范围：本官方网站积分商城兑换的所有礼品
一、实物礼品退换货政策:
积分商城兑换的任何一款实物礼品，若非礼品质量问题或发送（货）错误问题,将不接受退换货处理。
积分商城兑换的任何一款实物礼品，若存在质量问题、发送（货）错误问题，支持无条件退换货，我们承担退回运费。退换货前请与客服人员联系，确认具体情况及流程，并请注意： ● 退换货申请必须在客户收到商品之日起（以签收日期为准）15日内，超过时间不予退换货。
● 商品存在质量问题，如破损、污渍、版型问题、商品缺失等，不包含色差、客户选购尺码不合适等问题。
● 商品的外包装完好无损（包括包裹填充物及外包装箱或外包装袋），商品附件、吊牌、标签等齐全，商品保持官方网站积分商城兑换时的原质原样不影响产品二次销售。 ● 退款形式与支付形式一致，退回礼品的同时，支付该礼品的积分将返回至您的帐户。
以下情况不提供退换货服务：
● 超过退货规定时限的商品；
● 已经穿着过或洗涤过的商品；
● 已经修改或加工过的商品；
● 直接接触皮肤的内衣类商品；
● 商品吊牌、商标、票据等有缺失的商品；
● 少量色差问题商品（发错颜色商品除外）；
二、虚拟礼品退货政策:
积分商城兑换的任何一款虚拟类礼品，如优惠券，我司暂不接受退货处理，还请您的谅解。
